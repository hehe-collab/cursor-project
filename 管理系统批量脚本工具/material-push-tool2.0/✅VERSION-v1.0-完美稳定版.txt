🚀 使用方法
═══════════════════════════════════════════════════════════

### 1. 准备Excel数据
```
任务ID | 源账户ID | 素材关键词 | 目标账户ID列表 | 备注
------|---------|-----------|--------------|------
1     | C-XXX   | 桶礼盒波   | 7561360...   | 推送3个账户
2     | C-XXX   | 另一关键词 | 7605136...   | 推送3个账户
```

**支持格式**：
- 任务ID：数字（1、2、3）或文字（任务1、任务2）
- 目标账户：每行一个，自动清理前后分号

### 2. 启动工具
```bash
cd /Volumes/存钱罐/cursor项目/material-push-tool
npm start
```

### 3. 执行流程
1. 读取Excel任务
2. 启动浏览器，加载素材页面
3. 确认登录状态，按 Enter 开始
4. 自动创建10个标签页（并行模式）
5. 所有标签页同时执行推送任务
6. 任务完成，显示统计结果
7. 按 Enter 关闭浏览器

### 1. 并行架构设计
```javascript
// 分阶段执行，清晰可观察
```

### 2. 模式区分
```javascript
// 串行模式：需要刷新页面
executeTask(page, task, index, total, isParallel=false)

// 并行模式：不需要刷新页面
executeTask(page, task, index, total, isParallel=true)
```

### 3. 智能重试
```javascript
// 步骤级包装器
await withRetry(
  () => selectSourceAccount(page, accountId),
  '选择源账户',
  maxAttempts=3
);
```

### 4. 元素定位最佳实践
```javascript
// ✅ 优化后：直接使用最稳定的选择器
const option = page.locator('.el-select-dropdown__item, .el-autocomplete-suggestion li')
```
