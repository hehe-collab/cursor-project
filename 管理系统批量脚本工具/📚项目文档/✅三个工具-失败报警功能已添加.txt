**触发时机**：
- ✅ 任务执行失败时：立即发出 `beep` 声
- ✅ 批次完成时：如有失败，在详情显示时提醒
- ✅ 所有任务完成时：如有失败，再次提醒

**声音效果**：
- 🔔 系统默认提示音（beep一声）
- ⚡ 即时响应，无需等待
- 🎯 简单有效，零依赖

**实现方式**：
```javascript
process.stdout.write('\x07'); // 终端铃声
```

## 🔧 技术实现细节

### 终端铃声原理

```javascript
process.stdout.write('\x07');
```

- `\x07`：ASCII 控制字符 BEL（Bell）
- 历史：早期电传打字机的铃铛
- 现代：触发终端模拟器的提示音
- 兼容性：所有终端（macOS Terminal、iTerm2、VSCode等）
- 性能：零延迟，零依赖

### 数据结构

**失败任务对象**：
```javascript
{
  success: false,              // 失败标记
  taskId: "任务3",             // Excel中的任务ID
  error: "Timeout...",         // 错误详情
  taskLabel: "[批次1-任务3] 任务3",  // 完整标签（并行模式）
  batchIndex: 1,               // 批次序号（并行模式）
  taskNumber: 3                // 任务序号（并行模式）
}
```

### 代码位置

**发出铃声的位置**：
1. 任务失败时（catch块）- 立即提醒
2. 总结时（如有失败）- 再次提醒

**显示详情的位置**：
1. 批次完成时 - 批次级显示
2. 所有任务完成时 - 总结级显示
