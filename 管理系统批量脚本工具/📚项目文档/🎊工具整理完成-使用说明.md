### 方式 3: 快捷别名（高级用户）

在 `~/.zshrc` 文件末尾添加：

```bash
# TikTok 自动化工具快捷别名
alias tools='cd "/Volumes/存钱罐/cursor项目/管理系统批量脚本工具"'
alias ad2='cd "/Volumes/存钱罐/cursor项目/管理系统批量脚本工具/ad-automation-2.0" && npm start'
alias push2='cd "/Volumes/存钱罐/cursor项目/管理系统批量脚本工具/material-push-tool-2.0" && npm start'
alias link2='cd "/Volumes/存钱罐/cursor项目/管理系统批量脚本工具/promotion-link-tool-2.0" && npm start'
```

重新加载配置：
```bash
source ~/.zshrc
```

然后就可以简单地：
```bash
ad2      # 直接启动广告工具 2.0
push2    # 直接启动推送工具 2.0
link2    # 直接启动链接工具 2.0
```
