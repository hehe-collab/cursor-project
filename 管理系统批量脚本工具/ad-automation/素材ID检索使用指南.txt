📝 如何填写素材ID
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

步骤 1: 从网页复制素材ID
────────────────────────

在素材选择界面：
1. 找到"素材ID"输入框
2. 点击搜索按钮旁边的输入框
3. 从素材列表中复制需要的素材ID

【方法 A】逐个复制：
  - 复制第一个ID: 7601467303255392264
  - 复制第二个ID: 7601467161916375047
  - 复制第三个ID: 7601468503409344520

【方法 B】批量复制（推荐）：
  - 在素材列表中选中多行的素材ID列
  - Ctrl+C / Cmd+C 复制
  - 会得到如下格式：
    7601467303255392264
    7601467161916375047
    7601468503409344520


步骤 2: 粘贴到 Excel
────────────────────────

方法 A: 直接粘贴（推荐）✅
──────────────────────

1. 点击Excel中的"素材ID"单元格
2. 直接 Ctrl+V / Cmd+V 粘贴
3. Excel 会保留换行符，显示为：

   ┌──────────────────────┐
   │7601467303255392264   │
   │7601467161916375047   │
   │7601468503409344520   │
   └──────────────────────┘

4. 脚本会自动识别并正确处理 ✅


方法 B: 手动输入
──────────────────────

1. 点击Excel中的"素材ID"单元格
2. 输入第一个ID: 7601467303255392264
3. 按 Alt+Enter (Windows) 或 Option+Enter (macOS) 换行
4. 输入第二个ID: 7601467161916375047
5. 再按 Alt+Enter 换行
6. 输入第三个ID: 7601468503409344520


方法 C: 使用分号分隔（也支持）
──────────────────────────────

如果复制粘贴后Excel自动添加了分号，也没关系：
  
  7601467303255392264;7601467161916375047;7601468503409344520

或者：
  
  ;7601467303255392264;7601467161916375047;7601468503409344520;

脚本会自动清理首尾的分号和多余空格 ✅


🔍 脚本处理逻辑
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Excel 原始数据:
  ";7601467303255392264\n7601467161916375047;7601468503409344520;"

脚本自动处理：
  1. 统一换行符 (\r\n, \n, \r → \n)
  2. 去除首尾分号和空格
  3. 按换行符、分号、逗号分隔
  4. 去除每个ID的前后空格
  5. 过滤空字符串

结果：
  ['7601467303255392264', '7601467161916375047', '7601468503409344520']

脚本执行时：
  1. 设置分页为30条/页
  2. 在"素材ID"输入框填入ID（换行分隔）
  3. 点击搜索
  4. 等待结果加载
  5. 点击"本页全选"
  6. 点击确认

📞 常见问题
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Q1: 复制粘贴后有分号怎么办？
A1: 不用担心，脚本会自动清理。直接粘贴即可。

Q2: 可以用逗号分隔素材ID吗？
A2: 可以。脚本支持换行、分号、逗号等多种分隔符。

Q3: 素材关键词和素材ID都填了会怎样？
A3: 素材关键词优先，素材ID会被忽略。

Q4: 素材ID检索时会验证结果吗？
A4: 不会。使用ID检索时跳过内容验证，直接全选。

Q5: 后续行可以有不同的素材ID吗？
A5: 不可以。素材ID只能在首行填写，后续行会继承首行的设置。
    如果需要不同素材，建议使用不同的任务编号。

Q6: 如何在 Excel 中换行？
A6: 双击单元格进入编辑模式，按 Alt+Enter (Windows) 或 Option+Enter (macOS)
