📋 首次使用（macOS / Windows）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 安装 Node.js
   网址: https://nodejs.org/
   选择 LTS 版本，下载并安装

2. 安装依赖
   打开终端（macOS）或 PowerShell（Windows）：
   
   cd /Volumes/存钱罐/cursor项目/ad-automation    # macOS
   cd D:\projects\ad-automation                    # Windows
   
   npm install

3. 生成 Excel 模板
   npm run template
   
   会在 data/ 目录生成 tasks.xlsx

4. 填写任务数据
   打开 data/tasks.xlsx
   参考模板中的示例数据填写
   
   ⚠️ 重要：
   - 相同任务编号的行属于同一批次
   - 首行必须填写完整配置
   - 后续行只需填 账户ID 和 推广链接关键词

5. 运行脚本
   npm start
   
   执行流程：
   ① 浏览器自动打开
   ② 如需登录，手动登录
   ③ 关闭所有弹窗
   ④ 确认在批量工具页面
   ⑤ 按 Enter 开始执行


📦 跨设备迁移（macOS ↔ Windows）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【需要复制的文件】✅
  - config.js
  - index.js
  - create-template.js
  - package.json
  - lib/ 文件夹
  - data/ 文件夹
  - README.md
  - .gitignore

【不要复制的文件】❌
  - node_modules/    （新设备重新 npm install）
  - browser-data/    （登录状态，每个设备独立）
  - pw-browsers/     （浏览器文件，自动下载）

【迁移步骤】
  1. 复制上述✅文件到新设备
  2. 新设备上运行: npm install
  3. 首次运行需要重新登录


⚙️ 配置说明
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【可修改】config.js

  - username / password  → 登录账号密码
  - delay.short/medium/long → 操作延时（不建议改太小）
  - headless: true → 无头模式（不显示浏览器）

【Excel 必填字段】

  任务         → 任务编号（相同编号=同一批次）
  主体         → 主体名称（如 MAD）
  账户ID       → 数字ID（如 7561696938035413008）
  Pixel        → Pixel名称（如 IDNLow-X）
  项目名称     → 项目名称（如 父女情深-IDN）
  推广链接关键词 → 搜索关键词（每个账户独立）
  素材关键词   → 搜索素材
  标题         → 多个用逗号分隔（hsx-印尼-7,hsx-印尼-8）
  优化目标     → 价值 或 下单
  出价         → 数字
  预算         → 数字
  开始日期     → YYYY-MM-DD
  开始时间     → HH:MM:SS

【可选字段】

  年龄         → 18+ 或 25+（默认 18+）
  认证身份     → 留空=不选
  商品库       → 留空=不选
  商品         → 留空=不选
  启用         → true/false（默认 true）
  提交次数     → 1-10次（默认 1）


🐛 常见问题
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【问题1】npm install 很慢
  解决：npm config set registry https://registry.npmmirror.com
       npm install

【问题2】Windows 权限错误
  解决：以管理员身份运行 PowerShell

【问题3】找不到"设置"按钮
  解决：① 确认已关闭所有弹窗
       ② 确认在批量工具页面
       ③ 手动点击"设置"后按 Enter

【问题4】素材选择不正确
  解决：脚本会验证搜索结果
       如果不匹配会暂停提示
       检查 Excel 中的"素材关键词"

【问题5】标题找不到
  解决：脚本会自动滚动查找
       如果找不到会暂停
       手动选择后按 Enter

【问题6】登录状态丢失
  解决：删除 browser-data 文件夹
       重新运行，手动登录一次
