六、使用建议
--------------------------------------------------------------------

### 日常使用（推荐配置）

```javascript
// config.js
parallel: {
  enabled: true,           // ✅ 10并发
  maxConcurrent: 10,
},
validation: {
  enableScreenshot: true,  // ✅ 自动截图
  enableValidationLog: true, // ✅ 增强验证
},
```

**适合**：
- 中等批量任务（20-50个）
- 需要验证和证据的场景
- M4 Pro最佳性能发挥

---

### 测试场景

```javascript
// config.js
parallel: {
  enabled: false,  // 串行模式，方便观察
},
validation: {
  enableScreenshot: true,
  enableValidationLog: true,
},
```

**适合**：
- 测试新功能
- 调试问题
- 验证数据一致性

---

### 大批量场景（100+任务）

```javascript
// config.js
parallel: {
  enabled: true,
  maxConcurrent: 10,
},
validation: {
  enableScreenshot: false,  // 关闭截图节省空间
  enableValidationLog: false, // 关闭日志简化输出
},
```

**适合**：
- 大批量任务（100+）
- 脚本已充分测试
- 需要最快速度
