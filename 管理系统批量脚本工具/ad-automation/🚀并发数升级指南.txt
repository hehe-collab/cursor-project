六、风险与注意事项
--------------------------------------------------------------------

### 可能的问题

**问题1：CPU占用过高**
- 现象：风扇狂转，系统卡顿
- 解决：降低并发数

**问题2：内存压力**
- 现象：活动监视器显示黄色或红色
- 解决：降低并发数或关闭其他应用

**问题3：服务器限流**
- 现象：多个任务同时失败
- 解决：降低并发数或增加batchDelay

**问题4：浏览器卡顿**
- 现象：标签页响应慢，动画卡顿
- 解决：降低并发数

---

### 降级方案

**如果升级后不稳定**，立即改回：

```javascript
// config.js
parallel: {
  enabled: true,
  maxConcurrent: 5,  // ← 改回5
  batchDelay: 2000,
}
```

**5并发已经提升80%，完全够用！** ✅
