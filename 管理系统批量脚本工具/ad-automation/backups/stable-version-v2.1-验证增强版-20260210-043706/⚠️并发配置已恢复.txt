八、注意事项
--------------------------------------------------------------------

### 关于自动截图

在10并发模式下：
- 10个任务同时运行
- 每个任务5张截图
- 约50张截图会在短时间内生成

**空间占用**（10并发批次）：
- 10个任务 × 5张截图/任务 = 50张截图
- 约 10-25 MB/批次

**如需节省空间**，可以临时关闭截图：
```javascript
validation: {
  enableScreenshot: false,  // 关闭截图
  enableValidationLog: true, // 保留验证日志
}
```

---

### 关于验证日志

在10并发模式下，验证日志会交错输出。如果觉得混乱，可以：

**方案1：关闭验证日志**
```javascript
validation: {
  enableScreenshot: true,    // 保留截图
  enableValidationLog: false, // 关闭验证日志
}
```

**方案2：降低并发数**
```javascript
parallel: {
  maxConcurrent: 5,  // 降低到5并发
}
```
