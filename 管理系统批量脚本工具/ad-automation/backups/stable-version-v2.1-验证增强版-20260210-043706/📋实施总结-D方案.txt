二、新增功能
--------------------------------------------------------------------

### 1. 自动截图（5张/任务）

✅ 01-主体和账户选择.png
✅ 02-项目设置.png
✅ 03-广告组设置.png
✅ 04-提交前最终确认.png
✅ 05-提交成功.png

**保存位置**：./screenshots/{任务ID}/
**特点**：保存在项目目录（外置硬盘），不占用系统盘

---

### 2. 增强验证日志

✅ 账户选择验证
✅ 出价验证
✅ 预算验证

**格式**：
```
  📋 Excel配置: {字段} = {预期值}
  ✓ 实际执行: {实际值}
  🔍 是否一致: ✅/❌
```


三、配置项
--------------------------------------------------------------------

```javascript
// config.js - validation 配置段

validation: {
  enableScreenshot: true,        // 是否启用自动截图
  enableValidationLog: true,     // 是否启用增强验证日志
  screenshotDir: './screenshots', // 截图保存目录
  screenshotQuality: 80,          // 截图质量（0-100）
}
```

**可随时开关**：
- enableScreenshot: false → 关闭截图
- enableValidationLog: false → 关闭验证日志


四、使用方法
--------------------------------------------------------------------

### 正常使用（无需额外操作）

```bash
npm start
```

脚本会自动：
1. 在关键步骤自动截图
2. 在关键操作后验证数据
3. 发现不一致时立即警告

---

### 执行后检查

```bash
# 1. 查看截图
open ./screenshots/

# 2. 对比截图与Excel配置
# 3. 登录后台抽查实际数据
```

六、空间占用

**单个任务**：

**100个任务**：
- 500张截图：约 100-250 MB

**清理方法**：
```bash
# 删除7天前的截图
find ./screenshots -type f -mtime +7 -delete

# 或手动删除不需要的任务文件夹
```
