📋 更新内容
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✅ 问题：
   之前所有账户共享同一组标题（从首行读取），无法为每个账户设置不同的标题

✅ 解决方案：
   标题字段改为**账户级别**，每个账户可以有自己独立的标题


🔧 技术修改
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. lib/excel.js
   ─────────────
   - 将标题解析从 taskGroup 级别移至 account 级别
   - 每个账户的标题支持继承或独立设置：
     * 如果当前行有标题值 → 使用当前行的标题
     * 如果当前行标题为空 → 继承首行的标题
   - 添加账户标题验证（缺少标题的账户会被跳过）

2. lib/automation.js
   ───────────────────
   - 修改 setupAdGroup 函数，从 account.titles 读取标题
   - 注释说明："每个账户使用自己的标题"

3. test-excel.js
   ──────────────
   - 修改显示逻辑，在每个账户详情中显示标题
   - 修改验证逻辑，检查所有账户是否都有标题

4. README.md
   ──────────
   - 更新字段说明：标题支持逐行自定义
   - 更新继承规则：标题字段必须填写（每个账户独立）
   - 更新示例：展示不同账户使用不同标题


📊 Excel 表格填写示例
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【方式 1: 每个账户使用不同的标题】✅ 推荐

任务 | 主体 | 账户ID         | 标题              | ...
-----|------|---------------|-------------------|-----
  1  | MAD  | 7561696938... | hsx-7,hsx-8,hsx-9 | ...
  1  |      | 7532009013... | hmj-1,hmj-2,hmj-3 | ...
  1  |      | 7561696495... | ph-1,ph-2,ph-3    | ...
  1  |      | 7561696693... | wz-1,wz-2,wz-3    | ...

结果：
  - 账户 7561696938... 使用标题: hsx-7, hsx-8, hsx-9
  - 账户 7532009013... 使用标题: hmj-1, hmj-2, hmj-3
  - 账户 7561696495... 使用标题: ph-1, ph-2, ph-3
  - 账户 7561696693... 使用标题: wz-1, wz-2, wz-3


【方式 2: 部分账户共享，部分账户独立】

任务 | 主体 | 账户ID         | 标题              | ...
-----|------|---------------|-------------------|-----
  1  | MAD  | 7561696938... | hsx-7,hsx-8,hsx-9 | ... (首行标题)
  1  |      | 7532009013... |                   | ... (继承首行)
  1  |      | 7561696495... | ph-1,ph-2,ph-3    | ... (独立标题)
  1  |      | 7561696693... |                   | ... (继承首行)

结果：
  - 账户 7561696938... 使用标题: hsx-7, hsx-8, hsx-9 (自己的)
  - 账户 7532009013... 使用标题: hsx-7, hsx-8, hsx-9 (继承首行)
  - 账户 7561696495... 使用标题: ph-1, ph-2, ph-3 (自己的)
  - 账户 7561696693... 使用标题: hsx-7, hsx-8, hsx-9 (继承首行)


【方式 3: 所有账户共享同一组标题】

任务 | 主体 | 账户ID         | 标题              | ...
-----|------|---------------|-------------------|-----
  1  | MAD  | 7561696938... | hsx-7,hsx-8,hsx-9 | ... (首行标题)
  1  |      | 7532009013... |                   | ... (继承首行)
  1  |      | 7561696495... |                   | ... (继承首行)
  1  |      | 7561696693... |                   | ... (继承首行)

结果：
  - 所有账户都使用标题: hsx-7, hsx-8, hsx-9


🎯 字段说明
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

标题（H列）：
  ✅ 支持逐行填写，每个账户可以有不同的标题
  ✅ 多个标题用英文逗号分隔（如：hsx-7,hsx-8,hsx-9）
  ✅ 如果某一行为空，继承首行的标题
  ⚠️  首行必须填写标题（作为默认值）
  ⚠️  如果某个账户既没有自己的标题，首行也没有标题，该账户会被跳过

🚀 使用建议
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. ✅ 如果每个账户需要不同的标题：
   在每一行的"标题"列都填写具体的标题

2. ✅ 如果部分账户共享标题：
   首行填写共享标题，需要独立标题的账户单独填写

3. ✅ 如果所有账户共享标题：
   只在首行填写标题，其他行留空

4. ⚠️  推荐做法：
   在 Excel 中先设置好第一行的所有配置
   然后逐行填写账户ID、推广链接、标题
   其他字段（Pixel、出价、预算等）如果相同就留空
