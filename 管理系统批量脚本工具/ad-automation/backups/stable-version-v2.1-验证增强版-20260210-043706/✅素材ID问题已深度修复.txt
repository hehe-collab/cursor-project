🔍 修复的问题

【症状】
- 每个任务第一个账户都卡住
- 界面显示正确的素材，但脚本无法继续
- 广告组界面来回滑动
- 需要手动点击Enter

【根源】
1️⃣ catch块吞掉错误 → 重试失效
2️⃣ 弹窗没关闭 → 2个弹窗同时存在
3️⃣ 素材ID填入不稳定 → 第一次经常失败

────────────────────────────────────────────────────

✅ 修复方案（5个改进）

1. ✅ 重新抛出错误（6个函数）
   - selectMaterials
   - selectPromotionLink
   - selectTitles
   - selectOptimizationTarget
   - inputBidAndBudget
   - selectStartTime

2. ✅ 错误前关闭弹窗
   - 避免弹窗残留

3. ✅ 重试前清理弹窗
   - 检测并关闭已存在的弹窗

4. ✅ 增强填入稳健性
   - 填入内容验证
   - 填入失败重试（2次）
   - 触发多个事件（blur + change + input）
   - 增加等待时间

5. ✅ 增加搜索等待时间
   - 给搜索更多时间完成
