--------------------------------------------------------------------

### Step 1: 启用并行模式 ⚙️

**打开文件**：config.js（第40行）

**修改**：
```javascript

  enabled: true,          // ← 改这里！改为 true
  maxConcurrent: 5,       // 保持不变

}
```

**保存文件**（Cmd+S）
