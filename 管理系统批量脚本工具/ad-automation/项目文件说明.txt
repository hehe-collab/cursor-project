📁 项目根目录
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【核心文件】- 需要复制到其他设备

  ✅ package.json
     作用: Node.js 项目配置文件，定义依赖包和脚本命令
     包含: playwright, xlsx 等依赖
     命令: npm start, npm run template
  
  ✅ config.js
     作用: 脚本配置文件
     包含: 网站URL、登录账号、延时设置、浏览器配置
     可修改: username, password, delay 等参数
  
  ✅ index.js
     作用: 主程序入口
     功能: 读取Excel → 启动浏览器 → 执行任务
  
  ✅ create-template.js
     作用: Excel 模板生成器
     运行: npm run template
     生成: data/tasks.xlsx


【文档文件】- 需要复制到其他设备

  ✅ README.md
     作用: 完整的使用说明文档（Markdown格式）
     包含: 详细的功能说明、字段解释、跨设备迁移指南、常见问题
  
  ✅ 快速开始.txt
     作用: 简化版的快速开始指南（纯文本）
     包含: 首次使用步骤、跨设备迁移清单、常见问题
  
  ✅ 项目文件说明.txt
     作用: 本文件，说明项目结构和各文件用途
  
  ✅ .gitignore
     作用: Git 忽略文件列表
     忽略: node_modules, browser-data, pw-browsers 等


【辅助脚本】- 需要复制到其他设备

  ✅ 启动脚本.bat (Windows)
     作用: Windows 一键启动脚本
     功能: 检查环境 → 安装依赖 → 生成模板 → 启动脚本
     运行: 双击执行
  
  ✅ 启动脚本.sh (macOS/Linux)
     作用: macOS/Linux 一键启动脚本
     功能: 检查环境 → 安装依赖 → 生成模板 → 启动脚本
     运行: ./启动脚本.sh


📁 lib/ 文件夹（核心代码）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【需要复制到其他设备】

  ✅ lib/utils.js
     作用: 工具函数模块
     包含:
       - log()         日志输出（带时间戳和图标）
       - delay()       固定延时
       - shortDelay()  短延时（200ms）
       - mediumDelay() 中等延时（400ms）
       - longDelay()   长延时（800ms）
       - pauseForUser() 暂停等待用户手动处理
       - safeClick()   安全点击（失败时暂停）
       - safeType()    安全输入（失败时暂停）
  
  ✅ lib/excel.js
     作用: Excel 读取和解析模块
     功能:
       - 读取 data/tasks.xlsx
       - 列名映射（中文→英文）
       - 任务分组（按任务编号）
       - 配置继承（后续行继承首行）
       - 数据验证（检查必填字段）
     导出: readTasks(), printTaskSummary()
  
  ✅ lib/automation.js
     作用: Playwright 自动化核心逻辑
     包含:
       - launchBrowser()    启动浏览器
       - preparePage()      页面准备（等待用户登录）
       - dismissPopups()    关闭弹窗
       - selectEntity()     选择主体
       - selectAccounts()   选择账户
       - setupProject()     设置推广项目
       - setupAdGroup()     设置广告组
       - submitTask()       提交任务
       - executeTaskGroup() 执行单个任务组
       - 以及各个步骤的子函数（选择推广链接、素材、标题等）


📁 data/ 文件夹（任务数据）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【需要复制到其他设备】

  ✅ data/tasks.xlsx
     作用: 任务数据文件
     格式: Excel 文件，包含两个 sheet:
           - 任务列表: 实际的任务数据
           - 字段说明: 字段含义和示例
     生成: npm run template
     填写: 参考模板中的示例


📁 node_modules/ 文件夹（依赖包）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【不要复制到其他设备】

  ❌ node_modules/
     作用: Node.js 依赖包文件夹
     大小: 约 300-500 MB
     包含: playwright, xlsx, readline 等
     
     ⚠️  跨设备迁移时不要复制！
     原因: 
       1. 文件很大，复制很慢
       2. 不同操作系统的依赖包不兼容
       3. 每个设备应该独立安装依赖
     
     安装: 在新设备上运行 npm install


📁 browser-data/ 文件夹（浏览器数据）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【不要复制到其他设备】

  ❌ browser-data/
     作用: 浏览器用户数据目录
     包含:
       - 登录 Cookie（保存登录状态）
       - 浏览历史
       - 缓存文件
       - 用户偏好设置
     
     ⚠️  跨设备迁移时不要复制！
     原因:
       1. 数据是平台相关的（macOS vs Windows）
       2. 包含敏感的登录信息
       3. 直接复制可能导致登录失效
     
     处理: 每个设备首次运行时，手动登录一次，
           脚本会自动保存登录状态到本地


📁 pw-browsers/ 文件夹（Playwright 浏览器）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【不要复制到其他设备】

  ❌ pw-browsers/
     作用: Playwright 下载的 Chromium 浏览器文件
     大小: 约 200-300 MB
     版本: 与 Playwright 版本对应
     
     ⚠️  跨设备迁移时不要复制！
     原因:
       1. 文件很大
       2. 不同操作系统的浏览器不兼容
       3. npm install 时会自动下载
     
     安装: npm install 时自动下载
           或手动: npx playwright install chromium
