## 🎯 使用方法

### 1. 准备Excel数据
```bash
npm run template  # 生成模板
```

填写 `data/link-tasks.xlsx`：
- 任务ID
- 模板链接名称
- 剧名
- 免费/预览集数
- 新链接名称列表（每行一个）

### 2. 启动工具
```bash
npm start
```

### 3. 执行流程
1. 确认任务清单
2. 浏览器自动打开
3. 确认已登录
4. **多个任务同时执行**
5. 查看结果汇总
