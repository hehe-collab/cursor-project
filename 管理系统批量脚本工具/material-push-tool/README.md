# 🚀 素材推送自动化工具

TikTok广告素材跨账户批量推送自动化工具，基于Playwright浏览器自动化技术。

---

## ✨ 功能特性

- ✅ **批量推送**：一键推送素材到多个目标账户
- ✅ **自动翻页**：自动处理所有页面的素材
- ✅ **多任务支持**：支持多个推送任务批量执行
- ✅ **自动重试**：步骤级自动重试机制，提高成功率
- ✅ **Excel配置**：通过Excel表格配置任务，简单直观
- ✅ **详细日志**：实时显示执行进度和状态

---

## 📋 使用场景

### 典型场景

**需求**：将账户A的某类素材（如"桶礼盒波"）推送到账户B、C、D、E、F

**手动操作**：
1. 选择源账户A
2. 搜索关键词"桶礼盒波"
3. 全选第1页素材
4. 点击批量同步
5. 输入目标账户ID
6. 确定推送
7. 翻到第2页
8. 重复步骤3-6
9. 重复直到所有页面完成

**自动化后**：
- 配置Excel：源账户、关键词、目标账户
- 运行脚本：`npm start`
- 自动完成所有操作

## 🛠️ 安装和配置

### 1. 安装依赖

```bash
cd material-push-tool
npm install
```

### 2. 生成Excel模板

```bash
npm run template
```

这将在 `data/push-tasks.xlsx` 生成模板文件。

### 3. 配置推送任务

打开 `data/push-tasks.xlsx`，按照以下格式填写：

| 任务ID | 源账户ID | 素材关键词 | 目标账户ID列表 | 备注 |
|--------|----------|------------|----------------|------|
| 任务1 | C-MADxd-IDN-2-8W-11 | 桶礼盒波 | 7561360888110<br>7561360897493<br>7561360323361 | 推送到3个账户 |
| 任务2 | C-MADxd-IDN-2-8W-11 | 另一个关键词 | 账户1<br>账户2<br>账户3 | 推送到3个账户 |

**字段说明**：

- **任务ID**：任务的唯一标识（可选，默认为"任务1"、"任务2"...）
- **源账户ID**：素材来源账户的ID（必填）
- **素材关键词**：搜索素材的关键词（必填）
- **目标账户ID列表**：推送目标账户ID，**每行一个账户ID**（必填）
- **备注**：任务备注信息（可选）

**⚠️ 注意事项**：
- 目标账户ID列表中每行一个账户ID
- 前后的分号会自动清除
- 空行会自动过滤

## 🚀 使用方法

### 1. 测试Excel数据

```bash
npm test
```

这将验证Excel数据格式是否正确，并显示任务摘要。

### 2. 执行推送任务

```bash
npm start
```

**执行流程**：
1. 脚本会自动打开浏览器
2. 导航到素材管理页面
3. 提示您确认已登录（如未登录，请手动登录）
4. 按 Enter 开始执行
5. 自动完成所有推送任务
6. 显示执行结果
7. 按 Enter 关闭浏览器

## 📊 执行流程

```
启动浏览器
  ↓
读取Excel任务
  ↓
导航到素材页面
  ↓
对于每个任务：
  ├─ 选择源账户
  ├─ 搜索素材关键词

  ↓
显示汇总报告
```

## ⚙️ 配置说明

配置文件：`config.js`

### 主要配置项

```javascript
module.exports = {
  // 素材页面URL
  materialUrl: 'https://adminxyz.dramahub8.com/advertiseTools/material',
  
  // Excel数据文件路径
  excelPath: './data/push-tasks.xlsx',
  
  // 延迟配置
  delay: {
    afterPush: 2000,        // 推送后等待2秒（界面刷新）
    beforeNextPage: 1000,   // 翻页前等待1秒
    afterSearch: 1500,      // 搜索后等待1.5秒
  },
  
  // 重试配置
  retry: {
    enabled: true,
    maxAttempts: 3,         // 最多重试3次
    retryDelay: 1000,       // 重试间隔1秒
  },
};
```

## 🐛 故障排查

### 1. 找不到元素 / 超时错误

**原因**：页面加载慢或网络不稳定

**解决**：
- 增加 `config.timeout.default` 的值（默认30秒）
- 检查网络连接

### 2. 素材未选中

**原因**：页面状态异常

**解决**：
- 手动刷新页面
- 按照提示手动处理后按 Enter 继续

### 3. 推送失败

**原因**：目标账户不存在或权限不足

**解决**：
- 检查目标账户ID是否正确
- 确认账户权限

### 4. Excel读取错误

**原因**：Excel格式不正确

**解决**：
- 运行 `npm run template` 重新生成模板
- 检查必填字段是否填写
- 确保目标账户ID列表格式正确（每行一个）

## ❓ 常见问题

### Q1: 是否需要每次都登录？

**A**: 不需要。浏览器会保持登录状态（存储在 `browser-data/` 目录）。

### Q2: 可以同时推送多个关键词吗？

**A**: 可以。在Excel中配置多个任务即可。

### Q3: 如果某个任务失败了怎么办？

**A**: 脚本会暂停，提示您手动处理，处理后按 Enter 继续下一个任务。其他任务不受影响。

### Q4: 推送速度有限制吗？

**A**: 每次推送后会等待2秒，避免触发频率限制。

### Q5: 如何确认推送成功？

**A**: 
1. 查看终端日志（显示推送了多少个素材）
2. 手动登录后台查看推送任务状态
