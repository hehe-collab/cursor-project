### 1. 选择工具

- 需要批量创建广告 → `ad-automation-2.0`

- 需要复制推广链接 → `promotion-link-tool-2.0`


## 💡 实用技巧

### 技巧 1: 创建快捷方式

在 `~/.zshrc` 或 `~/.bashrc` 中添加：

```bash
# 快速导航到工具目录
alias ad2="cd '/Volumes/存钱罐/cursor项目/管理系统批量脚本工具/ad-automation-2.0'"
alias push2="cd '/Volumes/存钱罐/cursor项目/管理系统批量脚本工具/material-push-tool-2.0'"
alias link2="cd '/Volumes/存钱罐/cursor项目/管理系统批量脚本工具/promotion-link-tool-2.0'"

# 快速打开 Excel
alias ad-excel="open '/Volumes/存钱罐/cursor项目/管理系统批量脚本工具/ad-automation-2.0/data/ad-tasks-2.0.xlsx'"
alias push-excel="open '/Volumes/存钱罐/cursor项目/管理系统批量脚本工具/material-push-tool-2.0/data/push-tasks-2.0.xlsx'"
alias link-excel="open '/Volumes/存钱罐/cursor项目/管理系统批量脚本工具/promotion-link-tool-2.0/data/link-tasks-2.0.xlsx'"
```

重新加载配置：
```bash
source ~/.zshrc
```

然后就可以：
```bash
ad2           # 直接进入广告工具目录
ad-excel      # 直接打开广告任务 Excel
```

### 技巧 2: 一键启动脚本

创建 `~/启动广告工具.sh`：
```bash
#!/bin/bash
cd "/Volumes/存钱罐/cursor项目/管理系统批量脚本工具/ad-automation-2.0"

```

使用：
```bash
chmod +x ~/启动广告工具.sh
~/启动广告工具.sh
```

## 🐛 常见问题

### Q: 移动后工具还能正常运行吗？

A: ✅ 完全可以！所有工具使用相对路径，移动后不影响使用。

### Q: 浏览器数据会丢失吗？

A: ✅ 不会！每个工具的 `browser-data/` 文件夹一起移动，登录状态保留。

### Q: Excel 文件路径会有问题吗？

A: ✅ 不会！代码中使用相对路径 `./data/xxx.xlsx`，移动后仍然有效。

### Q: 如何在新路径下使用？

A: 只需要在新路径下运行命令即可，使用方法完全一样。
