## 💡 实用技巧

### 技巧 1: 创建快捷方式

在 `~/.zshrc` 或 `~/.bashrc` 中添加：

```bash
# 快速导航到工具目录
alias ad2="cd '/Volumes/存钱罐/cursor项目/管理系统批量脚本工具/ad-automation2.0'"
alias push2="cd '/Volumes/存钱罐/cursor项目/管理系统批量脚本工具/material-push-tool2.0'"
alias link2="cd '/Volumes/存钱罐/cursor项目/管理系统批量脚本工具/promotion-link-tool2.0'"

# 快速打开 Excel
alias ad-excel="open '/Volumes/存钱罐/cursor项目/管理系统批量脚本工具/ad-automation2.0/data/tasks-v2.0.xlsx'"
alias push-excel="open '/Volumes/存钱罐/cursor项目/管理系统批量脚本工具/material-push-tool2.0/data/push-tasks-v2.0.xlsx'"
alias link-excel="open '/Volumes/存钱罐/cursor项目/管理系统批量脚本工具/promotion-link-tool2.0/data/link-tasks-v2.0.xlsx'"
```

重新加载配置：
```bash
source ~/.zshrc
```

然后就可以：
```bash
ad2           # 直接进入广告工具目录
ad-excel      # 直接打开广告任务 Excel
```

### 技巧 2: 一键启动脚本

创建 `~/启动广告工具.sh`：
```bash
#!/bin/bash
cd "/Volumes/存钱罐/cursor项目/管理系统批量脚本工具/ad-automation2.0"

```

使用：
```bash
chmod +x ~/启动广告工具.sh
~/启动广告工具.sh
```
