# 新老剧区分方案记录

用于 C 组数据的新剧/老剧区分，当前采用**方式 A**。其他方式记录如下，后续需要时可切换。

---

## 方式 A（当前采用）· 剧名列表 = 新剧名单

- **规则**：剧名列表中的 = 新剧，C 组其余 = 老剧
- **优点**：无需新增配置，沿用现有剧名列表
- **缺点**：剧名列表同时承担「分析选中」和「新剧名单」两个角色

---

## 方式 B · 新增「新剧」列

- **实现**：在剧名列表 Excel 中增加一列「新剧」，勾选/填 1 表示新剧
- **优点**：新剧标记与「是否分析」解耦，可只分析部分剧同时标记新剧
- **缺点**：需修改 Excel 结构

---

## 方式 C · 新剧单独 sheet

- **实现**：剧名列表 Excel 中新增 sheet「新剧名单」，列出本周新剧
- **优点**：新剧名单与要分析的剧分开维护，更清晰
- **缺点**：配置稍复杂，需维护两个 sheet

---

## 方式 D · 配置文件指定

- **实现**：在 config.yaml 或单独配置文件中写 `新剧名单: [剧A, 剧B, ...]`
- **优点**：不改 Excel 结构
- **缺点**：维护不如 Excel 直观
