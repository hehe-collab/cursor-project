# 素材像素级复刻工具 (Auto Video Copier)

这是一款基于图像特征（pHash）和序列对齐技术的短剧素材全自动复刻工具。
主要用于：给定一段中文混剪广告素材，自动从印尼语全集视频中寻找对应的无字幕画面，并无缝拼接成一模一样的印尼版素材。

## 项目结构
- `src/`：核心源代码目录。
  - `main.py`：自动化工作流的主入口程序。
- `data/`：存放为全集视频建立的特征字典（JSON文件）。做到数据与代码分离，提高复用性。
- `input/`：请将您的“全集原剧.mp4”和“待复刻的中文素材.mp4”放入此目录。
- `output/`：最终生成的印尼版视频将保存在这里。

## 环境要求
- Python 3.9+
- FFmpeg 7.0+
- 依赖包请执行：`pip install -r requirements.txt`
